# RoadRead

Веб-застосунок для вивчення дорожніх знаків у форматі вікторини. Призначений для учнів, які готуються до іспиту з ПДР.

## Як працює

Гравцю показується опис знаку — потрібно вибрати правильний із чотирьох варіантів. За кожну правильну відповідь росте лічильник серії 🔥. Неправильний вибір скидає серію.

## Запуск

Відкрити `index.html` у будь-якому браузері. Інтернет потрібен лише для завантаження Tailwind CSS з CDN.

## Додавання знаків

У файлі `index.html` знайдіть масив `signsData` і додайте об'єкт:

```js
{
  number: "1.3",
  name: "Назва знаку",
  description: "Текст питання, який побачить учень.",
  image: `<svg ...>...</svg>`
}
```

SVG кожного знаку рендериться у Shadow DOM — конфлікти CSS-класів між знаками виключені.

## Технології

- Vanilla JS, без фреймворків
- Tailwind CSS (CDN)
- Shadow DOM для ізоляції SVG

## Структура файлу

```
index.html
├── tailwind.config      — кольори, анімації
├── <style>              — стилі компонентів
├── <header>             — HUD зі лічильником серії
├── <main>               — питання + сітка варіантів
├── <div#result-overlay> — оверлей правильної відповіді
└── <script>
    ├── signsData[]      — база знаків
    ├── buildRoundData() — логіка вибірки знаків
    ├── preloadNextRound() — фоновий prerender SVG
    └── handleChoice()   — обробка відповіді
```
